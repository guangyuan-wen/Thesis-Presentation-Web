IMAGES PLACEHOLDER DIRECTORY
================================

Place your images here with the following filenames:

  hero-bg.jpg      — Full-screen hero background image for the homepage
                     Recommended: a wide landscape/park photo, or a render of Hong Lim Park
                     Recommended size: 1920 x 1080px or larger

The system will automatically use the image once placed here.
If no image is found, the hero falls back to a dark gradient background.
