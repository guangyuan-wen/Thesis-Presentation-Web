THE PROBLEM

The Tool Gap in Design:
Fragmentation vs. Unity

⬡

Isolated Islands

Office collaboration is seamless — Docs, Sheets, Slides. Yet the master plan still lives in a single-user expert silo, invisible and inaccessible to those it affects most.

◎

Accessibility Gap

Professional CAD and BIM tools carry enormous technical barriers, preventing meaningful public participation and client engagement in landscape design decisions.

SITE LABORATORY

Speaker's Corner —
Hong Lim Park, Singapore

Democratic Context

Singapore's most symbolically democratic public space. If a park is to be redesigned through a democratic process, Hong Lim Park is the ultimate choice.

  • Supports high-pressure needs: rallies, speeches, daily recreation
  • Key elements: Outdoor Stage, Central Lawn, Shaded Perimeters
  • Located next to Clarke Quay MRT
Auditability

Every design decision must be traceable and defensible. The platform records every click as a structured coordinate set, creating a complete audit trail from public preference to final design.

  • 300 design sessions → 300 JSON files
  • Python aggregator → 1 consensus heatmap
  • Each dimension linked to a Rule ID
THE METHODOLOGY

Data Flow:
Preference to Places

01

HARVEST

Users enter the prototype via a simple link. Every asset placement is recorded as a structured coordinate set, enabling real-time data capture at scale.

→
02

EXPORT

The system generates a technical JSON list of coordinates: Asset_ID, Coord_X, Coord_Y, Area, Rotation, Cost — ready for immediate BIM / Rhino / AutoCAD integration.

→
03

AGGREGATE

Python scripts merge thousands of individual data points and calculate spatial density, transforming subjective choices into an objective consensus heatmap.

→
04

TRANSLATE

IF-THEN rules convert the heatmap into precise geometry for professional workflows. Every dimension in the final plan is mathematically mandated by the data.

design_rule.json 
IF (User_Dwell_Density > 0.8)
   AND (Solar_Radiation > 400W)
THEN [Add_Canopy_Radius = 6m]

// Every dimension is traceable to a Rule ID
// backed by the JSON dataset
CORE PRINCIPLES

Three Pillars of
Collaborative Landscape Design

◈

HUMAN

Human-led choices at every step. The platform empowers non-expert participants to make meaningful spatial decisions through an intuitive interface that requires zero technical knowledge.

Preference-Driven
⬡

DEMOCRATIC

Traceable public agency. Every participant's input is recorded and weighted equally. The aggregated heatmap becomes a mathematical mandate — not just a suggestion.

Auditable
◎

COMPUTABLE

Seamless data export to professional workflows. Structured JSON coordinates integrate directly with Rhino, BIM, and AutoCAD — bridging public participation with expert execution.

BIM-Ready
THE PLATFORM

LANDSCAPE PRO
Interactive Interface

A web-based collaborative design platform accessible via a simple link — no installation required. Users place landscape assets on a scaled site plan, generating structured coordinate data in real time.

  • Real-time budget meter with cost tracking
  • Asset library: Big Trees, Benches, Plazas, Ponds
  • 3 park style templates (Urban / Community / Nature)
  • Instant JSON export for professional workflows

FUTURE DEVELOPMENT

Scaling Beyond
the Pilot

Hong Lim Park is just the beginning. The vision: a city with 6 million planners continuously engaged in a collaborative conversation — redesigning Fort Canning Park, Marina Bay, and the entire Singapore Green Network.

01

Botany Module

Species growth metrics and biodiversity selection, integrating ecological performance data into every design decision.

02

Policy Engine

Real-time GNPR (Green Plot Ratio) and LRR compliance tracking, ensuring every public submission meets regulatory standards.

03

Economics Layer

Automated cost estimation and full lifecycle maintenance analysis, making the economic impact of design choices transparent.

04

BIM Integration

Direct live connection to BIM / Rhino-Inside, closing the loop between public preference and professional construction documentation.

Towards a Collaborative Dialogue

Transforming landscape architecture from an opaque expert process into an auditable, human-led collaborative workflow.
