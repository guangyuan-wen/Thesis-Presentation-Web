PROBLEM STATEMENT

The Tool Gap in Design:
Fragmentation vs. Unity

Shaping landscape architecture: the tools we use define what's possible.

⬡

Isolated Islands

Office work is seamless — Docs, Sheets, Slides. Yet the master plan still lives in a single-user expert silo. The process is opaque, inaccessible, and unaccountable to those it affects most.

◎

Accessibility Gap

Professional CAD/BIM tools carry enormous technical barriers. There is no lightweight, browser-based way for the public or clients to meaningfully participate in landscape design conversations.

THE SOLUTION

Remote Collaboration Platform:
Universal Accessibility

A web interface designed for remote participants — no installation, just a link.

01

Join the design conversation from any device. No software to install. No training required. Just open the link and start placing assets on the site plan.

THE LOGIC BRIDGE

Translating Preference to Rules:
IF-THEN Design Logic

Public spatial patterns are converted into CAD-exportable design rules. Every dimension in the final plan is mathematically mandated.

design_rules.json 
IF (User_Dwell_Density > 0.8)
   AND (Solar_Radiation > 400W)
THEN [Add_Canopy_Radius = 6m]

// Path width, bench spacing, canopy radius
// — each dimension is linked to a Rule ID
// — each Rule ID backed by the JSON dataset
◈
Full Auditability: The evidence chain runs from raw preference data → spatial density map → IF-THEN rule → final dimension in the CAD file.
HONEST ASSESSMENT

Limitations &
Mitigation Strategies

Every methodology has blind spots. Here's how this research addresses them.

Challenge
Mitigation
Sampling Bias

Online tools may exclude elderly users and those without digital access

→

Deploy physical tablets on-site to capture in-person participants alongside the digital cohort

Nonsensical Inputs

Users may place elements randomly, generating data that doesn't reflect genuine preference

→

AI-assisted outlier detection filters sessions that deviate significantly from all spatial logic

Technical Constraints

IF-THEN rule logic may inadvertently suppress creative or unconventional design solutions

→

Future versions will include a "Free Draw" mode to allow unconstrained input alongside the structured workflow
